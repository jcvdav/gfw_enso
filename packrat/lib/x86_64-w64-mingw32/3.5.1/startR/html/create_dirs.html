create_dirs {startR}R Documentation

Create directories

Description

Creates a set of standard directories: raw_data, data, rmd, docs, scripts

Usage

create_dirs(other_dirs = NULL)

Arguments

other_dirs

If needed, a character vector containing the names for extra directories.

Examples


## Not run: 
create_dirs() # This creates default directories

create_dirs(other_dirs = "figures") # Creates all default directories and a figures directory

## End(Not run)
